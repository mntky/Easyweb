# Easyweb
ちゃちゃっとwebサーバ立てたい

```
git clone https://github.com/mntky/webu
```

## Options

```
-a X.X.X.X	//IPAddress
-p XXXX		//Port
-t test		//Webpage text
```

## Example

```
ew							//localhost:8080
ew -a 192.168.1.1 -p 9000	//Listen on 192.168.1.1:9000
ew -a 192.168.1.1 -t test	//Listen on 192.168.1.1:8080 display test
```
